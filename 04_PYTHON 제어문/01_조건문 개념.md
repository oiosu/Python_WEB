##### ✔ IF 문 사용법 

> **기존** 비밀번호 = "1234"
>
> ```python
> origin_pass = "1234"
> ```
>
> **입력한** 비밀번호 = "1234"
>
> ```python
> input_pass = "1234"
> ```
>
> **만약** 비밀번호를 정확히 입력했으면 > 로그인 성공 
>
> ```python
> if origin_pass == input_pass:
> 	print("로그인 성공")
> ```
