## 2_연산(2)



#### ⭐⭐⭐비교연산 (조건문에서 많이 사용하기 때문에 정말 많이 중요하다)⭐⭐⭐

| 연산자(연산 기호) | 설명(왼쪽이 오른쪽보다) |
| :---------------: | :---------------------: |
|         >         |          크다           |
|         <         |          작다           |
|        >=         |       크거나 같다       |
|        <=         |       작거나 같다       |
|        ==         |          같다           |
|        !=         |         다르다          |



#### 논리연산 

| 연산자  |              설명              |
| :-----: | :----------------------------: |
| A and B |    A, B 모두 참이라면 True     |
| A or B  | A, B 중 하나라도 참이라면 True |
|  not A  |      A 가 참이라면 False       |



#### 멤버십 연산 

| 연산자(연산기호) | 설명(왼쪽이 오른쪽보에) |
| :--------------: | :---------------------: |
|        in        |     포함되어 있다.      |
|      not in      |   포함되어 있지 않다.   |
